# 🏆 Академия чемпионов

Веб-приложение для детей 6–10 лет, которое превращает ежедневные задания в игру. Ребёнок выполняет задания по спорту, учёбе, творчеству и другим направлениям — копит очки, растёт по уровням и обменивает баллы на реальные призы от родителей.

## Для кого

- **Ребёнок** — видит задания, отмечает выполнение, следит за прогрессом и наградами
- **Родитель** — добавляет задания, подтверждает выполнение, управляет призами (защита PIN-кодом)

## Что внутри

- 📋 **Планировщик** — задания по 6 категориям (спорт, учёба, творчество, здоровье, дом, чтение) с очками и повторением
- ✅ **Подтверждение** — ребёнок отмечает выполнение, родитель засчитывает очки
- 📈 **Прогресс** — недельный график, динамика, статистика по категориям
- 🏅 **Уровни** — 6 уровней от «Новичка» до «Легенды» + значки-достижения
- 🎁 **Призы** — общие и личные, за очки или за достижение цели
- 👨‍💼 **Родительский режим** — защита PIN, обзор всех детей, управление заданиями

## Запуск

Откройте `index.html` в браузере — установка и интернет не нужны. Данные хранятся локально.

## Стек

Vanilla JS · CSS · HTML · localStorage (без сборки, без зависимостей)
