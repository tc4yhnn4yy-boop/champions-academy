Планировщик задач для детей

Создать сайт, планировщик задач по развитию ребенка по разным направлениям. Например, спорт, учеба и т.п. с возможностью начисления баллов за каждое выполненное задание. В последующем эти балы по завершению отчетного периода могут быть монитезированы. Цель - приучить ребенка дисциплине, проявить интерес к саморазвитию. Стиль должен быть легкий, дружелюбный и интересный для ребенка. Можно применить формат геймификаций, в стиле интересующих тем для ребенка. Мои дети например, интересуются футболом, любят обсуждать матчи чемпионата мира, лиги чемпионов и самих футболистов. Пользоваться сайтом будут дети младшего возраста, от 6 до 10 лет. Сайт должен состоять из блоков непосредственного планировщика, мониторинга исполнения и полученных наград (баллов, монетизации).
